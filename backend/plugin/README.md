# Plugins bundle

Готовый пакет для установки Joomla находится в:

- `MaxBot/` — основной плагин `system/MaxBot` + встроенный `vmpayment/order`.

Как установить:

1. Заархивируй содержимое каталога `MaxBot` в ZIP.
2. Установи ZIP через Joomla Extension Manager.
3. Встроенный `order` установится автоматически из `bundled/vmpayment/order` через `script.php`.
